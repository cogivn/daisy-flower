## Overview

A fill-in template to map any use case through the 5 active SDLC stages at LITE tier. Use this when applying the [Founder Playbook](../03-playbooks/founder-playbook.md) or whenever you need to structure a new workflow.

**Time to complete**: 30-60 minutes
**Output**: A clear, step-by-step plan with quality gates

## Completion Checklist

### Before starting:
- [ ] Use case clearly scoped (small enough)
- [ ] Current baseline measured
- [ ] Frequency validated (worth optimizing)

### Stage 00 — FOUNDATION:
- [ ] Problem statement written
- [ ] Impact defined
- [ ] AI opportunity identified

### Stage 01 — PLANNING:
- [ ] Specific outcomes defined
- [ ] Success criteria measurable
- [ ] Timeline realistic

### Stage 02 — DESIGN:
- [ ] Flow mapped step-by-step
- [ ] Human vs AI roles clear
- [ ] Quality gates defined (2+ minimum)

### Stage 03 — INTEGRATE:
- [ ] Tools selected (max 3)
- [ ] Data flow between tools understood

### Stage 04 — BUILD:
- [ ] 5+ iterations completed
- [ ] Results measured and compared to baseline
- [ ] Pattern documented (if successful)
